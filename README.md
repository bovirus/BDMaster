# BDMaster

[![Linux Build](https://github.com/caoccao/BDMaster/actions/workflows/linux_build.yml/badge.svg)](https://github.com/caoccao/BDMaster/actions/workflows/linux_build.yml) [![MacOS Build](https://github.com/caoccao/BDMaster/actions/workflows/macos_build.yml/badge.svg)](https://github.com/caoccao/BDMaster/actions/workflows/macos_build.yml) [![Windows Build](https://github.com/caoccao/BDMaster/actions/workflows/windows_build.yml/badge.svg)](https://github.com/caoccao/BDMaster/actions/workflows/windows_build.yml)

A modern GUI for inspecting Blu-ray disc.

![BDMaster](src-tauri/icons/128x128@2x.png)

## Features

* Linux (x86_64) + MacOS (x86_64 + arm64) + Windows (x86_64)
* Drag and drop Blu-ray disc or folder
* Disc, playlist, stream, track information
* Integrate with [BetterMediaInfo](https://github.com/caoccao/BetterMediaInfo)
* Integrate with [MKVToolNix](https://mkvtoolnix.download/)
